# Minecraft 1.8.0 Portable (Batch Launcher) 🚀

Портативная версия чистого оригинального клиента Minecraft 1.8.0, которая запускается напрямую через один файл запуска. Все файлы игры, ваши настройки, миры, библиотеки и файлы графического движка хранятся внутри папки проекта. Сборку можно переносить на обычной флешке и запускать на любом компьютере без установки.

## ✨ Особенности сборки
* **Чистый клиент:** Сборка не содержит сторонних тяжелых лаунчеров, встроенной рекламы и лишнего мусора.
* **Кастомный никнейм:** Скрипт вежливо запрашивает имя игрока перед каждым новым стартом.
* **Выбор скина (модели):** Вы можете самостоятельно выбрать, за кого играть — классического Стива (широкие руки) или Алекс (узкие руки).
* **Полная портативность:** Скрипт автоматически переопределяет стандартные пути сохранения операционной системы. Все настройки и миры остаются внутри папки с игрой, не забивая компьютер.

## 🚀 Инструкция по запуску

1. **Скачайте** эту папку со сборкой игры к себе на компьютер или сразу на флешку.
2. Убедитесь, что на компьютере установлена стандартная программа **Java 8**, без которой Майнкрафт не сможет работать.
3. Найдите в главной папке запускающий файл **`запуск_игры.bat`** и откройте его.
4. В появившемся черном окне консоли напишите свой игровой **никнейм** без пробелов и нажмите клавишу **Enter**.
5. Выберите нужного персонажа, нажав цифру **1** или **2**, и снова нажмите клавишу **Enter**. После этого откроется привычное главное меню игры!

## ⚠️ Важные примечания
* **Выделение оперативной памяти:** Настройки запуска подобраны так, чтобы игра работала плавно и без зависаний даже на обычных домашних компьютерах и ноутбуках. По умолчанию выделено 2 Гигабайта памяти.
* **Разрядность системы:** Файлы в папке игры автоматически подстроятся под вашу операционную систему для правильного отображения графики, хорошего звука и точной работы клавиатуры с мышкой.
