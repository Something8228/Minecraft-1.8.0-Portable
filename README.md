Портативный чистый Minecraft 1.8.0Удобная портативная версия оригинального клиента игры, которая работает напрямую через один запускающий файл. Игру можно легко запустить с флешки или перенести на любой другой компьютер. Все ваши сохранения, настройки и текстуры будут всегда находиться внутри одной папки и не оставят никаких следов в операционной системе Windows.Особенности сборкиАбсолютная чистота: Сборка не содержит сторонних тяжелых лаунчеров, рекламы, водяных знаков и лишнего мусора.Выбор имени игрока: При каждом новом старте запускающий файл вежливо попросит вас ввести желаемый игровой никнейм.Выбор модели персонажа: Вы можете самостоятельно выбрать, за кого играть — классического Стива с широкими руками или Алекс с узкими руками.Полная портативность: Скрипт автоматически переопределяет стандартные пути сохранения операционной системы. Все настройки и миры остаются внутри папки с игрой, не забивая системные папки компьютера.Инструкция по установке и запускуСкачайте все файлы из этого репозитория на свой компьютер или флеш-накопитель.Убедитесь, что в вашей системе установлена актуальная версия среды выполнения Java 8.Откройте главную папку проекта и запустите файл с расширением .bat.В открывшемся черном окне консоли введите свой никнейм без использования пробелов и нажмите клавишу Enter.Выберите желаемую модель персонажа, нажав цифру 1 или 2, и снова нажмите Enter. Игра успешно запустится!Важные примечанияВыделение оперативной памяти: По умолчанию для стабильной работы игры выделено 2 Гигабайта оперативной памяти. Если вы планируете запускать сборку на очень старом или слабом компьютере, вы можете изменить это значение в свойствах запускающего файла на 1 Гигабайт.Разрядность системы: Файлы библиотек в папке нативов должны строго соответствовать разрядности установленной у вас Java. Если игра закрывается с ошибкой совместимости, убедитесь, что разрядность файлов и вашей Java совпадают (32 или 64 бита).
